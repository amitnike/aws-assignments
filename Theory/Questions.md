# AWS Theory Assignments

## 1. Cloud Computing Fundamentals and Service Models

**Question:** Explain the core characteristics of cloud computing according to NIST. Compare the three service models (IaaS, PaaS, SaaS) with specific AWS examples, highlighting how each model affects customer responsibilities and use cases.

## 2. AWS Global Infrastructure and Architecture Design

**Question:** Describe AWS Global Infrastructure components (Regions, Availability Zones, Edge Locations). Design a multi-region architecture for a global web application requiring high availability and disaster recovery. Justify your service choices and explain how your design addresses latency and resilience requirements.

## 3. AWS Shared Responsibility Model

**Question:** Explain the AWS Shared Responsibility Model, detailing security responsibilities for both AWS and customers. Analyze how these responsibilities shift across different service types (IaaS, PaaS, SaaS) with specific examples of security controls for each party.

## 4. Cloud Migration Strategies

**Question:** Analyze the "6 Rs" of cloud migration (Rehost, Replatform, Repurchase, Refactor, Retire, Retain). For each strategy, provide a business scenario where it's most appropriate, including challenges, benefits, and suitable AWS migration tools.

## 5. AWS Pricing and Cost Optimization

**Question:** Compare AWS pricing models (On-Demand, Reserved Instances, Savings Plans, Spot Instances). Create a cost optimization strategy for a medium-sized enterprise with mixed workloads, specifying recommendations for different workload types and relevant AWS cost management tools.

## 6. Identity and Access Management Strategy

**Question:** Explain AWS IAM components (Users, Groups, Roles, Policies) and the principle of least privilege. Design an IAM strategy for an organization with development, testing, and production environments, including cross-account access and security best practices.

## 7. AWS Storage Architecture

**Question:** Compare AWS storage services (S3, EBS, EFS, FSx, Storage Gateway) focusing on characteristics, use cases, and performance. Design a storage architecture for an enterprise requiring database storage, web content hosting, analytics storage, and archival solutions.

## 8. Serverless Architecture Design

**Question:** Analyze serverless computing benefits and limitations compared to traditional architectures. Design a serverless e-commerce platform architecture including product catalog, order processing, and analytics components using appropriate AWS serverless services.

## 9. AWS Networking and Performance Optimization

**Question:** Explain AWS networking components (VPC, subnets, gateways, endpoints). Design a secure, high-performance network architecture for a multi-tier application. Explain how CloudFront and Global Accelerator could enhance performance and user experience.

## 10. Monitoring and Incident Response

**Question:** Describe how AWS monitoring services (CloudWatch, CloudTrail, Config) work together for comprehensive observability. Design a monitoring and incident response strategy for a mission-critical application, including detection mechanisms, automated remediation, and response procedures for common failure scenarios.
